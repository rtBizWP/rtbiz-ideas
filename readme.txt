Great! There are several ways you can get involved to help make rtBiz Helpdesk better:

1. **Report Bugs:** If you find a bug, error or other problem, please report it! You can do this by [creating a new support ticket](https://rtcamp.com/premium-support) on rtCamp's Helpdesk.

== Installation ==

* You can download the plugin from [My Account](https://rtcamp.com/my-account/) section on our store. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* rtBiz Ideas is dependent on [rtBiz](https://wordpress.org/plugins/rtbiz/) plugin. So once you activate rtBiz Ideas, it will ask you to install & activate rtBiz plugin.
